Plugin overview

The purpose of this plugin is provide a linkage between Nucleus CMS and the phpBB forum software. This plugin provides automatic authentication for Nucleus, while related plugins provide other integration functionality.

Installation

Instalation of the plugin is a multi-step process, of which you are likely on step 4.

  1. Download the NP_phpBB.zip file, and upload the contents to the plugins directory of your nucleus instalation.
  2. Install the phpBB plugin, per its instructions
  3. From the Nucleus Plugins screen, install this plugin. It has no configuration options, so none need to be set.
  4. Update skins to use desired skin variables (listed below).

Skinvars

Member details skin variable (<%phpBBvar%>)

This skinvar is designed to replace the contents of the member details page with a redirect to the appropriate page in the phpBB instalation. If the member can't be found in the phpBB database or the phpBB plugin isn't working, it tries to fall back to an existing skin. To set up this skinvar, you will need to go through the following steps for each skin you wish to use the skinvar in.

  1. From the 'Skins' page (under layout) in the Nucleus admin area, clone the skin you wish to alter (clone link under the actions column.)
  2. rename the cloned skin from 'clone_skinname' to 'fallback/skinname'.
  3. Delete all skinparts from 'fallback/skinname' except the Member Details skinpart
  4. Alter the Member Details section of 'skinname' (not 'fallback/skinname') to contain '<%phpBBvar%>'
